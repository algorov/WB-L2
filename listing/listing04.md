Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Инициализируется канал, который работает с типом int.
Запускается другая горутина, которая 10 раз пытается записать в канал, но исходя из принципа
работы небуферезированных каналов, горутина переходит в состояние ожидания и никогда не проснется,
поскольку из канала никто не читает.

Произойдет deadlock.
```
